# Scalpel

Path of Exile's first ever fourth-party tool. An overlay to edit your filter, price check items, and tons more.

## Building from Source

```
npm install
npm run dev
```

## Official Releases

Pre-built releases are available at [scalpel.fourth.party](https://scalpel.fourth.party).
